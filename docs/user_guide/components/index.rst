Time Domain
^^^^^^^^^^^
In order to support dynamic modeling, the IDAES modeling framework has a time domain. This time
domain exists even for steady state models (single point in time). Please see the 
:ref:`time domain documentation<user_guide/components/time_domain:Time Domain>` for more
information.
